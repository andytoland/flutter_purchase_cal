# Purchase Calc - Flutter App

This is the mobile frontend for the Purchase Calc project.

## Build/Test Commands
- **Build APK**: `flutter build apk`
- **Build iOS**: `flutter build ios`
- **Test**: `flutter test`
- **Lint**: `flutter analyze`
- **Add Dependencies**: `flutter pub add <package>`

## Project Status
- Active development.
- Synchronous development required with API and UI apps.

> [!IMPORTANT]
> All three components (API, Frontend, and Flutter app) must be developed and updated synchronously to ensure compatibility.
